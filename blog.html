Important FAQ

What is the Document Object Model (DOM)?

An interface designed for online pages is called the Document Object Model (DOM) language. Programs can modify the page's representation in order to alter the document's style, content, and structure.

How do you select an element from the DOM?

getElementById: This method selects an element by its unique ID. Because IDs in a document are ensured to be distinct, they are one of the most efficient methods to retrieve a particular section.
getElementsByClassName: This method retrieves a dynamic HTML Collection of elements sharing the same class name

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is a technique used in JavaScript for more efficient event management by making use of the event bubbling stage rather than binding event listeners to specific individuals.It is useful for
Performance: Enhances performance by decreasing the number of event listeners,
Dynamic Content: Automatically manages events for newly added child elements.
Simpler Code: Makes the code asier to manage by reducing the number of event listeners.

How do you manipulate an element's attributes and styles using the DOM?

Accessing Elements: Select elements by using getElementById or querySelector.
Changing Attributes: Modify attributes by using setAttribute or directly via properties, like element.src.
Adding/Removing Attributes: Use setAttribute to add and removeAttribute to remove attributes.
Managing Classes: Use classList to add, remove, or toggle classes for style changes.
