Important FAQ

What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content. The DOM represents the document as nodes and objects; that way, programming languages can interact with the page.

How do you select an element from the DOM?

getElementById: This method selects an element by its unique ID. It is one of the most efficient ways to access a specific element, as IDs are guaranteed to be unique within a document.
getElementsByClassName: This method retrieves a live HTMLCollection of elements that share a specified class name. It is useful for selecting multiple elements at once.
getElementsByTagName: This method selects elements by their tag name and returns a live HTMLCollection. It is useful for selecting all instances of a particular tag.

What is event delegation in the context of the DOM, and why is it useful?

Event Delegation is a technique used in JavaScript to manage events more efficiently by leveraging the event bubbling phase. Instead of attaching event listeners to individual child elements, you attach a single event listener to a parent element. It is useful for
Performance: Reduces the number of event listeners, which can improve performance, especially with many child elements.
Dynamic Content: Automatically handles events for dynamically added child elements, eliminating the need to reattach listeners.
Simpler Code: Makes the code cleaner and easier to manage since you have fewer listeners to keep track of.

How do you manipulate an element's attributes and styles using the DOM?

Accessing Elements: Use methods like getElementById or querySelector to select elements.
Changing Attributes:Modify attributes with setAttribute or directly via properties, like element.src.
Adding/Removing Attributes: Use setAttribute to add and removeAttribute to remove attributes.
Manipulating Styles: Adjust styles using the style property (e.g., element.style.color = 'blue').
Managing Classes: Use classList to add, remove, or toggle classes for style changes.
